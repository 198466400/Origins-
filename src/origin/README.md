# ORIGIN — System Root

The System Root wires all subsystems together:

- Awareness
- Understanding
- Reflection
- Adaptation
- Creation
- Evaluation
- Cycle Controller

The OriginEngine:
- loads configuration
- injects dependencies
- runs the introspection cycle
- manages lifecycle (start/stop)
- ensures safe, continuous operation

This is the top-level orchestrator of the entire ORIGIN system.
